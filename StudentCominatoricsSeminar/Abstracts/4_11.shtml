Introduction to Zeilberger Algorithms for Proving Terminating Hypergeometric Identities

Moa Apagodu

Virginia Commonwealth University

Abstract

Every "proper-Hypergeometric" multisum/integral identity with a fixed number of summation or integration signs, possesses a short, computer-constructed proof. In this talk we will go over the basics of Zeilberg Methods of proof. At the end of this talk, if not already familiar, attandants will be able to use the many versions of the algorithm.
